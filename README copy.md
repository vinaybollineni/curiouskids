# Curious Kids Learning Center LLC — Website

A clean, responsive website for Curious Kids Learning Center LLC home daycare in Liberty Hill, TX.

## Files

```
curious-kids-website/
├── index.html      # Main HTML page
├── styles.css      # All styles
├── script.js       # Interactions & animations
└── README.md       # This file
```

## Deploy to GitHub Pages (Free Hosting)

1. **Create a new GitHub repository**
   - Go to github.com → New repository
   - Name it: `curious-kids-website` (or anything you like)
   - Set to **Public**
   - Click **Create repository**

2. **Upload your files**
   - Click **Add file → Upload files**
   - Drag all 4 files (index.html, styles.css, script.js, README.md)
   - Click **Commit changes**

3. **Enable GitHub Pages**
   - Go to your repo → **Settings → Pages**
   - Under **Source**, select **Deploy from a branch**
   - Branch: **main** / Folder: **/ (root)**
   - Click **Save**

4. **Your site is live!**
   - URL: `https://yourusername.github.io/curious-kids-website`
   - Takes 1–2 minutes to go live after saving

## Customization Tips

- **Update contact info:** Edit `index.html` — search for phone/email/address
- **Update hours:** Search for "8:30 AM" in index.html
- **Add your photo:** Replace the about card section with an `<img>` tag
- **Change colors:** Edit CSS variables at the top of `styles.css`
- **Add Google Maps:** Replace the address in the contact section with an embedded map iframe

## Connect a Custom Domain (Optional)

Once live on GitHub Pages:
1. Buy a domain (e.g. `curiouskidslearningcenter.com`) from Namecheap or Google Domains (~$12/year)
2. In GitHub Pages settings, add your custom domain
3. Follow the DNS instructions provided
